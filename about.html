'Li'

Linear Regression is a supervised machine learning algorithm in which the predicted output of the trained model is continuous and has a constant slope. It is used to predict values in a continuous range. The statistical software used in this research work is SPSS. Research work accuracy and loss are the dependent variables which are the output values. The analysis of the research work is done using Independent T-Test, which is used to compare the accuracy and loss of Linear Regression and Logistic Regression.

Logistic regression is a supervised machine learning classification algorithm. It is used to predict the probability of a Reduction of Network Traffic for Cloud Based Communication Department of CSE MTIET, Palamaner 10 target variable. It is named after the core function used in the algorithm that is the logistic function. The logistic function can also be called the sigmoid function which was developed by the statisticians. It is an S-shaped curve that takes input as any real valued number and maps it into a value between 0 and 1, but never at those limits. In this function e is the base of the natural algorithms
